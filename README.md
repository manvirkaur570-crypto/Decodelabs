# Data Classification Using AI

## Description

This project is a simple machine learning classification model built using Python. It loads a small dataset, splits it into training and testing data, and applies a classification algorithm to predict outcomes.
The goal of this project is to understand the basics of supervised learning, model training, and evaluation.

## Features

* Load dataset using sklearn
* Split data into training and testing sets
* Train a classification model
* Predict results using test data
* Calculate model accuracy

## Technologies Used

* Python
* Scikit-learn (sklearn)
* Pandas (optional)

## How to Run

1. Install required libraries:
pip install scikit-learn


2. Run the script:
python classification.py

## Output
* Model accuracy score
* Predicted values vs actual values

## Learning Outcome

* Understanding supervised learning
* Data splitting (train/test)
* Basic classification concepts in AI
